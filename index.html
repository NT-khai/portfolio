React

là một thư viện JavaScript mã nguồn mở do Facebook phát triển, chuyên dùng để xây dựng giao diện người dùng (UI) cho các ứng dụng web có tính tương tác cao.

Git

Git là một hệ thống quản lý phiên bản phân tán, được sử dụng phổ biến trong phát triển phần mềm để theo dõi và quản lý lịch sử thay đổi của mã nguồn.

GitHub

một nền tảng lưu trữ mã nguồn dựa trên web, sử dụng hệ thống quản lý phiên bản Git.

bootstrap

TailWind
